Deploy the backend

  1. Create a Web Service from this repo on Render or Railway.
  2. Build: pip install -r requirements.txt
  3. Start: python gov.py
  4. Environment: set OPENAI_API_KEY (optional), OPENAI_API_BASE, OPENAI_MODEL.
